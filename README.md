# AeroSecure 🔐

AeroSecure is an advanced Python-based cybersecurity tool that performs port scanning, service detection, OS fingerprinting, and basic vulnerability analysis.

Features:
- Fast & full scan modes
- Multi-threaded scanning
- Service & banner detection
- OS guessing
- Risk level analysis
- Vulnerability suggestions
- CLI-based professional tool

Built for learning cybersecurity and real-world networking concepts.
